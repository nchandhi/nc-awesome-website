### Technical Metrics (Model Performance)
- **Accuracy**: % of correct predictions
- **Precision**: % of positive predictions that are correct
- **Recall**: % of actual positives correctly identified
- **F1 Score**: Balance between precision and recall
- **Latency**: Response time (ms)

### Business Metrics (ROI)
- **Cost Savings**: $ saved per month
- **Time Savings**: Hours saved per week
- **Revenue Impact**: $ increase in sales
- **User Satisfaction**: CSAT/NPS scores
- **Adoption Rate**: % of users actively using solution
